# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email **security@ewei.io** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to release a patch within 7 days for critical issues.

## Scope

The following are in scope:

- Authentication bypass or API key leakage in the SDK
- Injection vulnerabilities in request construction
- Dependency vulnerabilities in production dependencies
- Logic errors that could cause unintended gas sponsorship

## Disclosure

We follow coordinated disclosure. We will credit reporters in the release notes unless anonymity is requested.
